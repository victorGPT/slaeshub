## §1 术语定义

| 术语 | 定义 |
|------|------|
| **BD** | 使用本系统的业务拓展人员 |
| **客户** | BD 名下管理的交易用户，以 UID 唯一标识 |
| **7天贡献** | 交易所过去滚动7日从该客户获得的盈利。见 PRD_首页_MVP.md §1 |
| **滚动7日** | 以当日为终点、向前回溯 7 个自然日的时间窗口，非自然周。见 PRD_首页_MVP.md §1 |
| **变化%** | 过去7日指标值 vs 过去8周滚动均值的差值比率，正值绿色、负值红色 |
| **同步时间** | 数据最后一次更新的时间，展示在页面顶部右侧 |
| **分页** | 每页展示 20 条客户，超出时显示翻页控件 |

## §2 模块概述

### §2.1 定位

客户列表是 BD 查看和管理名下所有客户的核心页，按7天贡献排序展示全量客户，并通过跟进状态圆点直观感知哪些客户有待跟进事件。

### §2.2 目标平台

移动端 Web App（iOS/Android 浏览器 PWA，设计宽度 393px）。

### §2.3 前置条件

BD 已登录系统，客户数据已完成初始化加载。

### §2.4 MVP 范围

**做**
- 顶部导航栏（页面标题 + 数据同步时间）
- 搜索栏（客户名称 / UID 快速检索）
- 客户列表（按7天贡献排序，可切换升/降序）
- 列表表头（含 Tooltip 说明）
- 客户行（UID + 姓名 + 7天贡献）
- 分页控件（每页 20 条）
- 底部 Tab 导航栏

**不做**
- 收藏功能
- 多条件组合筛选
- 自定义列显示
- 批量操作

## §3 功能详述

### §3.1 顶部导航栏

#### 功能说明

固定在页面顶部，展示页面标题和数据同步时间。

#### 页面结构

| 位置 | 元素 | 说明 |
|------|------|------|
| 中间 | 页面标题「客户」| 主文案 |
| 右侧 | 同步时间 | 格式「HH:MM 同步」，灰色小字 |

---

### §3.2 搜索栏

#### 功能说明

BD 可通过客户姓名或 UID 快速检索，检索时客户列表隐藏，结果在输入框下方下拉展示，点击后跳转至对应客户详情页。

#### 触发方式

点击搜索框，键入字符后实时触发检索。

#### 页面结构

| 元素 | 状态 | 说明 |
|------|------|------|
| 搜索输入框 | 默认 | 展示占位文案「搜索客户名称或UID」 |
| 搜索输入框 | 输入中 | 右侧出现清除按钮，客户列表隐藏 |
| 清除按钮（×）| 有内容时显示 | 点击后清空输入，恢复客户列表 |
| 下拉结果列表 | 有匹配结果 | 至多 20 条，每行含姓名 + UID + 资沉 |
| 下拉结果列表 | 无匹配结果 | 展示「未找到匹配客户」|
| 下拉结果列表 | 输入为空 | 不显示，客户列表恢复 |

#### 字段说明（下拉结果每行）

| 字段 | 说明 |
|------|------|
| 客户姓名 | 主标识 |
| UID | 灰色次要标识 |
| 资沉 | 客户资产展示值，辅助识别客户 |

#### 操作结果

- 输入字符 → 客户列表隐藏，下拉列表实时更新
- 点击结果行 → 跳转客户详情页
- 点击清除 / 按 ESC → 清空输入，关闭下拉，客户列表恢复

#### 边界条件

| 场景 | 处理 |
|------|------|
| 输入为空 | 下拉不出现，客户列表正常展示 |
| 匹配超过 20 条 | 只展示前 20 条，不提示截断 |
| 无匹配 | 展示「未找到匹配客户」|

---

### §3.3 客户列表

#### 功能说明

以两列表格形式（客户 + 7天贡献）展示 BD 名下所有客户，默认按7天贡献降序排列，点击表头可切换升/降序。

#### 触发方式

进入页面时自动渲染，默认7天贡献降序。

#### 页面结构

| 元素 | 说明 |
|------|------|
| 表头行 | 客户 / 资沉（Tooltip）/ 7天贡献（可点击切换升降序 + Tooltip）|
| 数据行 | 每行一名客户，见字段说明 |
| 分页控件 | 总数 > 20 时显示：上一页 / 当前页/总页数 / 下一页 / 总人数 |

#### 字段说明（数据行）

| 字段 | 说明 |
|------|------|
| UID | 客户唯一标识，字重加粗 |
| 客户姓名 | UID 下方小字展示；无姓名时不显示 |
| 资沉数值 | K/M/B 格式，金色；无数据时显示「—」|
| 资沉变化% | 过去7日资沉 vs 过去8周滚动均值，正值绿色、负值红色；无数据时不显示 |
| 7天贡献数值 | K/M/B 格式，正值绿色、负值红色；无数据时显示「—」|
| 7天贡献变化% | 过去7日贡献 vs 过去8周滚动均值，正值绿色、负值红色；无数据时不显示。**注：原型阶段用 `netContribution.wow`（周环比）占位，正式上线需后端提供8周均值字段** |

#### 表头 Tooltip 说明文案

| 列 | Tooltip 文案 |
|----|-------------|
| 资沉 | 资沉：客户过去滚动7日持仓资产均值（USDT）／变化%：过去7日资沉 vs 过去8周滚动均值 |
| 7天贡献 | 7天贡献：交易所过去滚动7日从该客户获得的盈利／变化%：过去7日贡献 vs 过去8周滚动均值 |

#### 排序规则

- 默认按7天贡献（`netContribution.total`）降序
- 点击表头切换升/降序
- 切换后重置到第 1 页

#### 分页规则

- 每页展示 20 条
- 总数 ≤ 20 条时不显示分页控件
- 翻页后列表滚动回顶部

#### 操作结果

- 点击数据行 → 跳转至对应客户详情页
- 点击表头「7天贡献」→ 切换升/降序
- 点击表头 ? 图标 → 展示 Tooltip，点击浮层外区域关闭

#### 边界条件

| 场景 | 处理 |
|------|------|
| 客户总数 ≤ 20 | 不显示分页控件 |
| 搜索词无匹配 | 展示「未找到匹配的客户」|
| 资沉数据缺失 | 显示「—」，变化%不显示 |
| 7天贡献数据缺失 | 显示「—」，变化%不显示 |
