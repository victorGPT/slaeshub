## §1 术语定义

| 术语 | 定义 |
|------|------|
| **BD** | 使用本系统的业务拓展人员 |
| **客户** | BD 名下管理的交易用户，以 UID 唯一标识 |
| **资沉** | 客户过去滚动7日持仓资产均值（USDT）。见 PRD_首页_MVP.md §1 |
| **7天贡献** | 交易所过去滚动7日从该客户获得的盈利。见 PRD_首页_MVP.md §1 |
| **变化%** | 过去7日值 vs 过去8周滚动均值的差值比率，正值绿色、负值红色 |
| **系统标签** | 系统根据客户排名自动生成的标签（资沉排名 / 利润排名 / 盈亏排名），仅前100名显示 |
| **手动标签** | BD 在档案 Tab 手动为客户添加的标签，可增删 |
| **来源标签** | 进入详情页时携带的跳转来源标识（事件 / 列表 / 搜索） |
| **业务线** | 客户参与的交易产品类别（双仓合约 / 超级杠杆 / 理财 / 金融卡 / 云算力 / Mini合约 / 现货） |
| **昵称** | BD 在档案 Tab 为客户手动设置的备注名称，本地存储 |
| **联系方式** | BD 在档案 Tab 为客户记录的沟通渠道及账号信息 |
| **笔记** | BD 在档案 Tab 或 FAB 为客户记录的文字备忘，按创建时间倒序展示 |

## §2 模块概述

### §2.1 定位

客户详情页是 BD 查看单一客户完整信息的核心页，汇聚客户资产指标、事件跟进、档案记录三大维度，是 BD 日常沟通和决策的信息中枢。

### §2.2 目标平台

移动端 Web App（iOS/Android 浏览器 PWA，设计宽度 393px）。

### §2.3 前置条件

BD 已登录系统；URL 参数包含有效的客户 UID（`?uid=xxx`）。

### §2.4 MVP 范围

**做**
- 顶部导航栏（返回 + 页面标题）
- Header 卡片（身份信息 + 标签 + 资沉 + 7天贡献）
- 资产 Tab（资沉分布 / 月交易盈亏 / 周冲提 / 7天贡献明细）
- 事件 Tab（该客户事件列表，只读展示，点击卡片进入事件详情页）
- 档案 Tab（昵称编辑 / 联系方式管理 / 笔记）
- FAB 笔记快捷录入
- 底部 Tab 导航栏

**不做**
- 充提指标（Header 中已移除）
- 推迟事件功能（已移除）
- 事件 Tab 新建 / 标记完成 / 主动沟通操作（纯展示，操作在事件详情页完成）
- 收藏操作入口（MVP 暂不在详情页提供）
- 导出 / 分享客户信息

## §3 功能详述

> 阅读约定：每个功能模块按以下顺序描述——**功能说明 → 触发方式 → 页面/弹窗结构 → 字段说明 → 操作结果 → 错误/空状态**。

### §3.1 顶部导航栏

#### 功能说明

固定吸顶，包含返回按钮、页面标题和 Header 卡片。

#### 页面结构

| 位置 | 元素 | 说明 |
|------|------|------|
| 左侧 | 返回按钮 | 点击返回上一页 |
| 中间 | 页面标题「客户详情」| |
| 右侧 | 空占位 | 与左侧返回按钮对称 |

---

### §3.2 Header 卡片

#### 功能说明

展示客户核心身份信息和两项关键指标（资沉 + 7天贡献），吸顶固定，便于 BD 在任意 Tab 快速查阅。

#### 页面结构

| 行 | 内容 |
|----|------|
| Row 1：身份行 | UID（可复制）+ 昵称 + 来源标签（可选）|
| Row 2：标签行 | 系统标签（排名类）+ 手动标签 + 添加标签按钮 |
| Row 3：核心指标 | 左：资沉；右：7天贡献（各含变化%角标）|
| 底部 | 数据更新时间（右对齐，小字）|

#### 字段说明

| 字段 | 说明 |
|------|------|
| UID | 客户唯一标识；右侧有复制图标，点击复制到剪贴板，图标短暂变为 ✓ |
| 昵称 | BD 在档案 Tab 设置的备注名，未设置则不显示 |
| 来源标签 | 跳转来源：事件（蓝）/ 列表（灰）/ 搜索（灰）；无来源参数时不显示 |
| 系统标签 | 资沉 #N / 利润 #N / 盈亏 #N，仅排名前100才显示对应标签 |
| 手动标签 | BD 自定义，每个标签有删除按钮（×）|
| 添加标签按钮 | 打开标签 Sheet（见 §3.2.1）|
| 资沉 | 客户过去滚动7日持仓资产均值（USDT），金色大字；变化% = 过去7日 vs 过去8周滚动均值 |
| 7天贡献 | 交易所过去滚动7日从客户获得的盈利；正值绿色、负值红色；变化% = 过去7日贡献 vs 过去8周滚动均值（**注：原型阶段用 `wow` 周环比占位，上线需后端提供8周均值字段**）|
| 数据更新时间 | 格式「更新于 M月D日 HH:MM」|

#### Tooltip 说明文案

| 字段 | Tooltip |
|------|---------|
| 资沉 | 客户过去滚动7日持仓资产均值（USDT）；变化% = 过去7日资沉 vs 过去8周滚动均值 |
| 7天贡献 | 7天贡献：交易所过去滚动7日从该客户获得的盈利／变化%：过去7日贡献 vs 过去8周滚动均值 |

#### 操作结果

- 点击 UID → 复制到剪贴板，图标变 ✓，1.2 秒后恢复
- 点击标签 × → 删除该手动标签
- 点击「添加标签」→ 打开标签 Sheet
- 点击 ? 图标 → 展示对应 Tooltip，点击浮层外关闭

#### 边界条件

| 场景 | 处理 |
|------|------|
| 无昵称 | 昵称不显示，不占位 |
| 无来源参数 | 来源标签不显示 |
| 排名超出前100 | 对应系统标签不显示 |
| 7天贡献无数据 | 显示「--」|
| 资沉无数据 | 显示「--」|

#### §3.2.1 添加标签 Sheet

**功能说明**：BD 为客户添加预设或自定义标签。

**触发方式**：点击 Header 标签行的「添加标签」按钮。

**弹窗结构**：从底部弹出的 Sheet，包含预设标签区和自定义输入区。

**字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| 预设标签 | 多选 pill | 系统提供若干预设选项，点击即添加 |
| 自定义标签 | 文本输入 | 最多 10 字；点击「添加」写入 |

**操作结果**：标签写入本地存储，Header 标签行实时更新；点击 Sheet 外区域关闭。

### §3.3 资产 Tab

#### 功能说明

展示客户在各业务线的资产分布、交易盈亏、充提流向及7天贡献明细，帮助 BD 理解客户的资产结构和贡献来源。

#### 触发方式

点击 Tab Bar「资产」，或进入详情页时默认激活。

#### 子模块列表

| 子模块 | 标题 | 核心展示 |
|--------|------|---------|
| §3.3.1 | 资沉分布 | 7 业务线横向条形图 + 总资沉 |
| §3.3.2 | 周交易盈亏 | 4 品种横向条形图 + 周盈亏总额 |
| §3.3.3 | 周冲提 | 8 周双柱图（流入/流出）+ 周均净流入 |
| §3.3.4 | 7天贡献明细 | 8 项明细横向条形图 + 总贡献 |

---

#### §3.3.1 资沉分布

**核心指标**：客户过去滚动7日持仓资产均值（USDT），与8周均值对比变化%，和平台均值对比。

**业务线明细**（7项）：双仓合约 / 超级杠杆 / 理财 / 金融卡 / 云算力 / Mini合约 / 现货

**字段说明**

| 字段 | 说明 |
|------|------|
| 主值 | `c.fMean`，USDT 格式 |
| 变化% | 当前值 vs 过去8周滚动均值 |
| 平台均值对比 | 该客户 vs 全平台 BD 客户均值 |
| 各业务线值 | `depositDist.{业务线}` |

---

#### §3.3.2 周交易盈亏

**说明**：客户过去滚动7日各品种持仓盈亏，正值表示客户亏损（对平台有利）。

**品种明细**（4项）：双仓合约 / 超级杠杆 / Mini合约 / 现货

**字段说明**

| 字段 | 说明 |
|------|------|
| 主值 | 过去滚动7日盈亏，绿正红负 |
| 变化% | 当前7日值 vs 过去8周滚动均值 |

---

#### §3.3.3 周冲提

**说明**：过去8周每周的资金流入 / 流出估算（基于周持仓变化推算，仅供参考），以双柱图呈现趋势。

**字段说明**

| 字段 | 说明 |
|------|------|
| 主值 | 过去7周基线周均净流入 |
| 变化% | 本周净流入 vs 周均净流入 |
| 柱图 | 8周，绿柱=流入，红柱=流出；本周高亮，历周透明 |

---

#### §3.3.4 7天贡献明细

**说明**：交易所过去滚动7日从该客户获得的净贡献，按收入项和成本项展示明细。

**明细项**（8项）

| 类别 | 项目 |
|------|------|
| 收入（绿）| 合约 / 杠杆 / Mini合约 / 现货 / 金融卡 / 云算力 |
| 成本（红）| 理财利息 / 返佣 |

**字段说明**

| 字段 | 说明 |
|------|------|
| 主值 | `netContribution.total`，绿正红负 |
| 变化% | `netContribution.wow`（周环比，**注：原型占位，上线需8周均值字段**）|
| 各明细项 | `netContribution.{futures/leverage/mini/spot/card/cloud/savingsInterest/rebate}` |

---

### §3.4 事件 Tab

#### 功能说明

以只读方式展示该客户名下的所有事件（含已完成），不提供新建或操作功能。点击卡片跳转至事件详情页（follow-up-thread）进行后续处理。

#### 触发方式

点击 Tab Bar「事件」。

#### 页面结构

| 元素 | 说明 |
|------|------|
| 事件卡片列表 | 按创建时间降序；已完成事件标题置灰加删除线 |
| 空状态 | 无事件时展示「暂无事件」|

#### 事件卡片字段

| 字段 | 说明 |
|------|------|
| 事件类型 badge | 颜色规则同事件中心（见 PRD_事件中心_MVP.md §3.2）|
| 来源 badge | BD / 系统 |
| 创建时间 | 格式「M月D日 HH:MM」|
| 事件标题 | 进行中正常显示；已完成置灰 + 删除线 |
| 备注（note）| 有备注时展示在标题下方 |

#### 操作结果

- 点击事件卡片 → 跳转至对应事件详情页（`follow-up-thread.html?fuId=xxx`）

#### 边界条件

| 场景 | 处理 |
|------|------|
| 无事件 | 展示「暂无事件」|
| 已完成事件 | 卡片标题置灰加删除线 |

### §3.5 档案 Tab

#### 功能说明

BD 管理客户的备注昵称、联系方式和工作笔记。

#### 触发方式

点击 Tab Bar「档案」。

---

#### §3.5.0 基本信息（只读）

档案 Tab 顶部展示客户基本信息，不可编辑。

| 字段 | 说明 |
|------|------|
| KYC 等级 | 客户实名认证等级 |
| VIP 等级 | 客户 VIP 级别 |

> **原型说明**：当前为静态占位数据，上线需由后端提供。

---

#### §3.5.1 昵称编辑

**说明**：BD 为客户设置备注名称，设置后在 Header、事件列表等处同步显示。

**操作**：点击昵称旁的编辑图标 → 输入框激活 → 输入后点击保存。

**边界条件**：保存空值则清除昵称；Header 实时更新。

---

#### §3.5.2 联系方式管理

**说明**：BD 记录客户的沟通渠道账号，支持多条，可逐条删除。

**支持渠道**（7种）：Telegram / WhatsApp / 微信 / Discord / 电话 / 邮件 / 其他

**字段说明（每条）**

| 字段 | 说明 |
|------|------|
| 渠道标签 | 下拉选择 |
| 值 | 对应渠道的账号 / 号码 / 地址 |
| 删除按钮 | 点击删除该条 |

**操作**：点击「添加」→ 弹出表单（选渠道 + 输入值）→ 保存写入本地存储。

**边界条件**：渠道和值均必填，否则「保存」按钮不可点击。

---

#### §3.5.3 笔记

**说明**：BD 为客户记录文字备忘，按创建时间倒序排列，支持快速新增。

**页面结构**

| 元素 | 说明 |
|------|------|
| 笔记时间轴列表 | 每条含创建时间 + 文字内容 |
| 新增按钮 | 打开笔记输入抽屉 |
| 空状态 | 「还没有笔记」|

**操作**：点击「新增」/ FAB → 打开输入框 → 输入后保存 → 写入本地存储，列表顶部新增。

---

### §3.6 FAB 笔记快捷录入

#### 功能说明

固定在底部导航栏上方的浮动按钮，方便 BD 在任意 Tab 快速录入笔记，无需切换到档案 Tab。

#### 触发方式

点击右下角浮动笔记图标。

#### 行为规则

- 向下滚动超过 10px → FAB 隐藏
- 向上滚动超过 5px → FAB 重新出现
- 点击 FAB → 打开笔记录入抽屉（同 §3.5.3 新增入口）
