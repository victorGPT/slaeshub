# 客户详情页（当前实现）

> 固定头部 + 3 Tab 架构，10 个模块。对应原型 `client-detail.html`。

---

## 1. 页面架构

```
┌─────────────────────────────┐
│  固定头部（始终可见）          │
│  ├ 模块 1: 客户身份           │
│  ├ 模块 2: 资沉核心指标       │
│  └ 模块 3: 概览判断           │
├─────────────────────────────┤
│  Tab 栏: [事件] [资产] [档案] │
├─────────────────────────────┤
│  Tab 内容（可滚动）           │
└─────────────────────────────┘
```

切换 Tab 时头部不动，保持身份和资沉始终可见。

---

## 2. 固定头部（3 个模块）

### 模块 1：客户身份

| 元素 | 说明 |
|------|------|
| 头像 | 根据客户名自动生成（ui-avatars），金色边框 |
| 客户名 | 16px 加粗 |
| 状态标签 | 流失（红）/ 增长（绿）/ 稳定（灰），圆角 pill |
| UID | 灰色小字 |
| 来源标签 | URL 参数 `from` 控制，可选值：事件 / 客户列表 / 搜索 |

来源标签帮助 BD 理解"我为什么进到这个页面"——从事件来的聚焦事件处理，从客户列表来的聚焦主动维护。

### 模块 2：资沉核心指标

| 指标 | 展示 |
|------|------|
| 本周资沉 | **页面最大数字**（24px 加粗），回答"盘子多大" |
| vs 8 周均 | 灰色小字 `vs 8周均 $XX万`，回答"变化了多少" |
| 本周流入 | 绿色，`流入 $X.X万` |
| 本周流出 | 红色，`流出 $X.X万` |
| 异常告警 | 当本周流出 > 均值流出 × 2 时，显示红色提示 `▼ 流出是均值的 X.X 倍` |

**已移除的指标：**
- ~~趋势稳定度（R²）~~
- ~~近期趋势百分比（%/周）~~

资沉变化通过流入/流出双向数据直观展示，不再使用抽象统计指标。

### 模块 3：概览判断

| 元素 | 说明 |
|------|------|
| 状态标签 | 与模块 1 状态一致，再次强调当前判断 |
| 一句话备忘 | 输入框，输入即保存到 `localStorage('memo_' + uid)` |

备忘是 BD 的私人记忆外挂。不需要格式模板，一句话够用。

## 3. Tab 1：事件

### 模块 4：当前事件

- 展示该客户所有未处理事件（排除 `sessionStorage('processed_events')` 中已标记的）
- 每条事件：事件类型 + 子类型 + 状态 pill + 金额 + 时间
- 点击跳转 `event-thread.html?id=xxx`
- 无事件时：绿色勾 + "暂无待处理事件"

### 模块 5：跟进记录

- 时间线样式（左侧竖线 + 圆点）
- 人工记录：蓝色圆点
- 系统记录：灰色圆点
- 每条记录：日期 + 执行人 + 内容
- 顶部"+ 添加记录"入口（金色文字）
- 最新在最上面

---

## 4. Tab 2：资产

### 模块 6：资金分布

5 条业务线资沉占比：
1. 理财
2. 合约保证金
3. 现货钱包
4. 金融卡
5. 云算力

每行：业务名 + 金额 + 百分比 + 进度条。按金额降序排列。前两名进度条金色，其余蓝色。

### 模块 7：业务参与

6 条业务线参与状态：
1. 双仓合约
2. 超级杠杆
3. Mini合约
4. 理财
5. 金融卡
6. 云算力

已参与：白色文字 + 绿色勾。未参与：灰色文字 + "未参与"标签。顶部显示"已参与 X 条业务线"。

### 模块 8：资沉流动

双向柱状图，**默认展开**（非折叠）：
- 上半部分：流入柱（绿色），按周展示
- 零线分隔
- 下半部分：流出柱（红色），按周展示
- X 轴标签：W2 - W8（7 周）
- 图例：绿色方块 = 流入，红色方块 = 流出

### 模块 9a：综合净贡献

**默认折叠**，点击展开：
- 折叠态：标题 + 总额（金色大数字）+ 展开箭头
- 展开态：
  - 周环比（绿色正/红色负）
  - 各模块条形图：
    - 收入（绿色）：合约、杠杆、Mini合约、金融卡、云算力
    - 成本（红色）：理财利息、返佣

### 模块 9b：客户近期盈亏

- 近 7 天总盈亏：20px 大数字（盈绿亏红）
- 8 周周均盈亏：13px 辅助数字
- 按业务拆分条形图：双仓合约、超级杠杆、Mini合约、现货

---

## 5. Tab 3：档案

### 模块 10a：BD 标签

- 可编辑标签列表，蓝色 pill 样式
- 点击标签上的 × 删除
- "+ 添加标签"按钮（虚线边框），点击弹出 prompt 输入
- 持久化到 `localStorage('bd_tags_' + uid)`

### 模块 10b：备注

- 自由文本 textarea，深色背景
- 输入即保存到 `localStorage('bd_note_' + uid)`
- placeholder："添加备注..."

---

## 6. 硬规则

1. **本周资沉是页面最大数字** — 固定头部始终可见，切 Tab 不丢
2. **资沉变化用流入/流出展示** — 不用趋势稳定度、不用回归斜率
3. **异常用倍数提示** — "流出是均值的 X 倍"比"流出偏差 +120%"直观
4. **综合净贡献默认折叠** — 大多数场景不需要，深入分析时再展开
5. **资沉流动直接展开** — 资金流向是判断客户状态的核心依据
